- [fd](https://github.com/sharkdp/fd)
- [fzf](https://github.com/junegunn/fzf)
- git
- curl or wget
- unzip
- GNU tar
- gzip
- nodejs(for lsp)
- python3

## tmux

- prefix set \<C-Space\>
- \<prefix\>r reload tmux config
- \<prefix\>I Install tmux plugins

## zsh

### Installation

1. copy .zshrc to $HOME
2. source .zshrc
3. excute `chsh -s $(which zsh)`

### Command line tools

- fzf
- tldr
